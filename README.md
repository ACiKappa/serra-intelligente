# Serra Intelligente – Sistema IoT con Analisi Dati

Questo progetto implementa una serra intelligente basata su ESP32, un server Node.js
per l’analisi dati e una dashboard React per la visualizzazione e la configurazione.

L’obiettivo è fornire un sistema modulare, configurabile e scalabile,
capace di analizzare temperatura e umidità, generare classificazioni giornaliere,
correlazioni e confronti tra soglie di default e soglie personalizzate dall’utente.
