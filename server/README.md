# Server

Questa cartella contiene il codice backend del progetto.  
L’obiettivo è gestire la ricezione dei dati dai dispositivi IoT, salvarli nel database e fornire API per il frontend.

## Contenuti previsti
- endpoint per ricezione dati
- funzioni di analisi (statistiche, correlazioni, classificazioni)
- gestione soglie e configurazioni utente
- API REST per la dashboard
- connessione al database

## Note
La struttura e le funzionalità possono evolvere in base alle esigenze del progetto.
