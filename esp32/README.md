# ESP32

Questa cartella contiene il codice per il microcontrollore ESP32.  
Il dispositivo legge i sensori e invia i dati al server.

## Contenuti previsti
- lettura sensori (temperatura, umidità)
- invio dati tramite WiFi (MQTT/HTTP)
- configurazioni di base del dispositivo

## Note
La logica può essere adattata in base ai sensori disponibili e al tipo di comunicazione scelto.
