# Frontend

Questa cartella contiene l’interfaccia utente sviluppata in React.  
La dashboard permette di visualizzare i dati raccolti e le analisi generate dal server.

## Contenuti previsti
- grafici temperatura/umidità
- visualizzazione analisi giornaliere e in tempo reale
- pannello configurazione soglie
- funzioni di confronto tra soglie default e personalizzate

## Note
L’interfaccia può essere ampliata o semplificata in base al tempo disponibile e agli obiettivi del progetto.
